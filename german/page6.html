Die vorgehensweise der ausarbeitung der
reproduktion einer zeichbnung als unikat

Die Vorgehensweise der reproduktion einer Zeichnung als Unikat

ist bis ins kleinste Detail auf der DVD erläutert. Falls Sie keine

Möglichkeit haben die Kopie von der DVD qualitativ hochwertig

auszudrucken, lassen Sie sie lieber in einer Druckerei oder einem

guten Digitalkopierstudio ausdrucken, wo man Ihnen ohne

Probleme die Kopie oder Zeichnung auf hochwertigem Papier

(welcher, nach Möglichkeit, zwecks besseren Haftens der

Pastellfarbe, nicht glatt sein sollte) ausdrucken wird.

Häufig gibt es in Fotokopierstudios kein solches Papier, das Sie

benötigen, so dass es ratsam ist, dass Sie es selbst beschaffen.

Dammit die Reproduktion qualitativ hochwerig ist, und damit die

Zeichnung bis ins kleinste Detail sauber bleibt, ist es nötig alle

Hauptteile (wie auf der DVD gezeigt) mit hölzernen Pastellfarben

auszumalen.

Warum mit hölzernen Pastellfarben?

Sehr kleine Teile der Zeichnung können Sie nur mit gut gespitzter

Spitze einer Holzfarbe (Pastellfarbe) ausmalen.

Wenn Sie die Zeichnung qualitativ hochwertig und nach ihrem

Wunsch mit Pastellfarben ausgemalt haben, entfernen Sie sorgfältig

den Überschuss an Farbe, was ebenfalls auf det DVD gezeigt ist.

Nach Entfemen das Überschusses an Pastellfarbe, können Sie mit/p>

das Ausmalen mit trockenen Pastellfarben beginnen

Damit das Ausmalen mit trockenen Pastellfarben einfach und

leicht von statten geht und damit Sie auf der Zeichnung die

Farbübergänge einer Farbnuance in guter Qualität erhalten,

benutzen Sie ebenfalls ein Wattepad zum Entfernen von

Schminke, insbesondere, wenn Sie mit kleineren Zeichnungen

arbeiten. Falls die Zeichnung größer ist, benutzen Sie ein

Schwämmchen, denn ein Schwämmchen absorbiert die trockene

Pastellfarbe in seine Struktur besser als ein Wattepad. Beginnen

Sie immer mit der hellsten Farbe, damit das Mischen von

Pastellfarbe richtig verläuft, denn Sie können dunkle Farbe nicht

so einfach heller machen, wie Sie helle Farbe dunkler machen

können.
